# ETLit

A project to **E**xtract data from different Sources, **T**ransform the data as neccessary and then **L**oad them into another System form further processes.

The configuration can be done in the *config/config.json* file.


---
## Content
Planned Functions for this tool:

### Extraction:
- MSSQL Database
- Flat file csv
- gevis/BC API

### Transformation:
- Hook to call custom python functions

### Load
- MSSQL Database
- Flat file csv
- d3 BusinessObjects
